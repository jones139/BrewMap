OSM Brew Map Tag Queries

This page identifies entries in the OSM database that might be breweries, distilleries or wineries based on the name tag, or if they are tagged as vineyards. There are quite a few exceptions to try to avoid false positivies (like parks called 'Brewer Fields'). The query that produces this page can be seen at BrewMap.cfg if you are interested in suggesting improvements!

Please use the 'browse' links to check if these should be tagged as industry=brewery or craft=brewery, then edit them. The data for the page is updated every 10 minutes so you can check back a bit later to make sure your updated entries have disappeared from here and appeared on the main BrewMap map.

Note that the edit links do not work at the moment. I do not know why - they use the same syntax as you get on the OSM 'browse' page for the element.
